## Overview

This project applies **Data Envelopment Analysis (DEA)**, a non-parametric linear programming technique, to measure the **technical efficiency** of 124 countries using World Bank data for the year **2014**.

DEA identifies efficient and inefficient Decision Making Units (DMUs — in this case, countries) by constructing a best-practice frontier. Two models are estimated:

- **CRS** (Constant Returns to Scale) — also known as the CCR model (Charnes, Cooper & Rhodes, 1978)
- **VRS** (Variable Returns to Scale) — also known as the BCC model (Banker, Charnes & Cooper, 1984)

From these, **Scale Efficiency (SE)** is derived as:

$$SE = \frac{TE_{CRS}}{TE_{VRS}}$$

The analysis is **input-oriented**: it measures by how much inputs can be reduced while holding output constant.

## How to Run

1. Place `2nd Assignment.xlsx` on your Desktop (or update `setwd()` in the script to your preferred path).
2. Open `DEA_method.r` in RStudio.
3. Run the script. It will:
   - Load and prepare the dataset
   - Compute VRS and CRS technical efficiency scores
   - Calculate scale efficiency
   - Print summary statistics
   - Produce a histogram of scale efficiency
   - Print peer groups under VRS and CRS

```r
library(dplyr)
library(readxl)
library(Benchmarking)

setwd("C:\\Users\\User\\Desktop")
data <- read_excel("2nd Assignment.xlsx")

# Inputs
x <- cbind(data[,4], data[,5], data[,6])   # energy, labor, GFCF

# Output
y <- as.matrix(data[,3])                    # GDP

# DEA models
TEvrs <- dea(x, y, RTS="vrs", ORIENTATION="in")
TEcrs <- dea(x, y, RTS="crs", ORIENTATION="in")

# Scale efficiency
SE <- TEcrs$eff / TEvrs$eff
```

## Key Findings

- **5 countries achieved full scale efficiency (SE = 1):** China, Luxembourg, Norway, Switzerland, and the United States. These are the only countries operating at the Most Productive Scale Size (MPSS).
- The **overall mean scale efficiency is 0.397**, indicating that on average countries need to significantly reduce their inputs to be fully efficient.
- **Greece** has a scale efficiency of 0.47 (above average), needing a ~53% modification of inputs. Under VRS it is 7.8% inefficient; under CRS it is 56.7% inefficient.
- **Bottom performers** (Togo, Haiti, Kyrgyz Republic, Tajikistan, Benin) have scale efficiencies below 0.037, implying they need to modify inputs by over 96%.
- **Tajikistan** is a notable case: it is fully VRS-efficient (TE = 1.0) yet has a very low scale efficiency (SE = 0.031), explained entirely by its low CRS efficiency.

---

## Conclusion

The DEA analysis reveals a massive efficiency gap between leading and lagging countries. The gap is driven primarily by low CRS technical efficiency in the worst-performing nations. The analysis provides actionable benchmarks: less-efficient countries can look to their peer countries (those on the efficient frontier) as performance targets. Future research could extend this to a longitudinal study, comparing country efficiency across multiple years.
